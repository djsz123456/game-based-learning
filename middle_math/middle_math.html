1. 公式/定理速查

1.1 代数

1.1.1 因式分解公式

完全平方公式
(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²
两个数的和（或差）的平方，等于它们的平方和，加上（或减去）它们的积的2倍。
例：(2x + 3)² = 4x² + 12x + 9
平方差公式
a² - b² = (a + b)(a - b)
两个数的平方差，等于这两个数的和与这两个数的差的积。
例：9x² - 16 = (3x + 4)(3x - 4)
立方和公式
a³ + b³ = (a + b)(a² - ab + b²)
两个数的立方和，等于这两个数的和乘以它们的平方和减去它们的积。
例：x³ + 8 = (x + 2)(x² - 2x + 4)
立方差公式
a³ - b³ = (a - b)(a² + ab + b²)
两个数的立方差，等于这两个数的差乘以它们的平方和加上它们的积。
例：8x³ - 27 = (2x - 3)(4x² + 6x + 9)

1.1.2 数列通项/求和公式

等差数列
通项公式：aₙ = a₁ + (n - 1)d
求和公式：Sₙ = n(a₁ + aₙ)/2 = na₁ + n(n - 1)d/2
a₁是首项，d是公差，n是项数。
例：等差数列首项a₁=2，公差d=3，第5项a₅=2 + 4×3=14，前5项和S₅=5×(2+14)/2=40
等比数列
通项公式：aₙ = a₁·qⁿ⁻¹
求和公式：Sₙ = a₁(1 - qⁿ)/(1 - q) (q ≠ 1)
a₁是首项，q是公比，n是项数。
例：等比数列首项a₁=2，公比q=3，第4项a₄=2×3³=54，前4项和S₄=2×(1-3⁴)/(1-3)=80

1.1.3 指数/对数运算法则

指数运算法则
aᵐ·aⁿ = aᵐ⁺ⁿ
aᵐ/aⁿ = aᵐ⁻ⁿ
(aᵐ)ⁿ = aᵐⁿ
(ab)ⁿ = aⁿ·bⁿ
a>0，m、n为实数。
例：2³·2⁴=2⁷=128
对数运算法则
logₐ(MN) = logₐM + logₐN
logₐ(M/N) = logₐM - logₐN
logₐMⁿ = n·logₐM
logₐb = logₑb / logₑa（换底公式）
a>0且a≠1，M>0，N>0。
例：log₂(8×4)=log₂8 + log₂4=3+2=5

1.2 几何

1.2.1 勾股定理

勾股定理
a² + b² = c²
在直角三角形中，两条直角边的平方和等于斜边的平方。
例：直角三角形的两条直角边分别为3和4，斜边c=√(3²+4²)=5

1.2.2 三角函数诱导公式

三角函数诱导公式
sin(π/2 - α) = cosα
cos(π/2 - α) = sinα
sin(π - α) = sinα
cos(π - α) = -cosα
用于将任意角的三角函数转化为锐角的三角函数。
例：sin(150°)=sin(180°-30°)=sin30°=1/2

1.2.3 面积/体积公式

三角形面积
S = 1/2·底·高
S = 1/2·ab·sinC（a、b为两边，C为夹角）
两种计算三角形面积的方法。
例：底为4，高为3的三角形面积S=1/2×4×3=6
圆的面积和周长
面积：S = πr²
周长：C = 2πr
r为圆的半径。
例：半径为5的圆，面积S=π×5²=25π，周长C=2π×5=10π
圆柱的表面积和体积
表面积：S = 2πr² + 2πrh
体积：V = πr²h
r为底面半径，h为高。
例：底面半径为2，高为5的圆柱，体积V=π×2²×5=20π
圆锥的表面积和体积
表面积：S = πr² + πrl
体积：V = 1/3·πr²h
r为底面半径，h为高，l为母线长。
例：底面半径为3，高为4的圆锥，体积V=1/3×π×3²×4=12π
球的表面积和体积
表面积：S = 4πr²
体积：V = 4/3·πr³
r为球的半径。
例：半径为3的球，体积V=4/3×π×3³=36π

1.2.4 解析几何

两点距离公式
d = √[(x₂ - x₁)² + (y₂ - y₁)²]
计算平面上两点(x₁,y₁)和(x₂,y₂)之间的距离。
例：点(1,2)和(4,6)之间的距离d=√[(4-1)²+(6-2)²]=5
直线斜率公式
k = (y₂ - y₁)/(x₂ - x₁) (x₂ ≠ x₁)
计算通过两点(x₁,y₁)和(x₂,y₂)的直线的斜率。
例：通过点(1,2)和(4,6)的直线斜率k=(6-2)/(4-1)=4/3
直线方程
点斜式：y - y₁ = k(x - x₁)
斜截式：y = kx + b
一般式：Ax + By + C = 0
k为斜率，b为截距。
例：斜率为2，截距为3的直线方程：y=2x+3
圆的方程
标准式：(x - a)² + (y - b)² = r²
一般式：x² + y² + Dx + Ey + F = 0
(a,b)为圆心坐标，r为半径。
例：圆心在(2,3)，半径为4的圆的方程：(x-2)²+(y-3)²=16

1.3 函数

1.3.1 一次函数

一次函数 y = kx + b

定义域：R（全体实数）

值域：R

单调性：k>0时，单调递增；k<0时，单调递减

图像：直线，k为斜率，b为截距

例：y=2x+3，斜率为2，截距为3，单调递增

1.3.2 二次函数

二次函数 y = ax² + bx + c (a ≠ 0)

定义域：R

值域：a>0时，[4ac-b²/4a, +∞)；a<0时，(-∞, 4ac-b²/4a]

单调性：a>0时，在(-∞, -b/2a)上递减，在(-b/2a, +∞)上递增；a<0时相反

对称轴：x = -b/2a

顶点：(-b/2a, 4ac-b²/4a)

例：y=x²-4x+3，a=1>0，对称轴x=2，顶点(2,-1)，值域[-1,+∞)

1.3.3 反比例函数

反比例函数 y = k/x (k ≠ 0)

定义域：(-∞,0)∪(0,+∞)

值域：(-∞,0)∪(0,+∞)

单调性：k>0时，在(-∞,0)和(0,+∞)上分别递减；k<0时，在(-∞,0)和(0,+∞)上分别递增

对称中心：原点(0,0)

例：y=2/x，k=2>0，在(-∞,0)和(0,+∞)上递减

1.3.4 指数函数

指数函数 y = aˣ (a>0且a≠1)

定义域：R

值域：(0,+∞)

单调性：a>1时，单调递增；0

图像：过点(0,1)

例：y=2ˣ，a=2>1，单调递增，过点(0,1)

1.3.5 对数函数

对数函数 y = logₐx (a>0且a≠1)

定义域：(0,+∞)

值域：R

单调性：a>1时，单调递增；0

图像：过点(1,0)

例：y=log₂x，a=2>1，单调递增，过点(1,0)

1.3.6 三角函数

正弦函数 y = sinx

定义域：R

值域：[-1,1]

周期性：2π

奇偶性：奇函数

单调递增区间：[2kπ-π/2, 2kπ+π/2]

单调递减区间：[2kπ+π/2, 2kπ+3π/2]

对称中心：(kπ, 0)

对称轴：x = kπ + π/2

余弦函数 y = cosx

定义域：R

值域：[-1,1]

周期性：2π

奇偶性：偶函数

单调递增区间：[2kπ-π, 2kπ]

单调递减区间：[2kπ, 2kπ+π]

对称中心：(kπ+π/2, 0)

对称轴：x = kπ

正切函数 y = tanx

定义域：{x | x ≠ kπ + π/2, k∈Z}

值域：R

周期性：π

奇偶性：奇函数

单调递增区间：(kπ-π/2, kπ+π/2)

对称中心：(kπ/2, 0)

2. 易错点/易混点总结

易错点：分式分母不为0

错误：1/(x-1) 中x为任意实数

错误原因：忽略了分母不能为0的限制

正确解法

正确：x-1 ≠ 0 → x ≠ 1

提醒：分式有意义的条件是分母不为0

易错点：二次根式被开方数非负

错误：√(-x) 中x为任意数

错误原因：忽略了被开方数必须非负的限制

正确解法

正确：-x ≥ 0 → x ≤ 0

提醒：二次根式√a中a ≥ 0

易错点：0指数幂的底数限制

错误：0⁰ = 1

错误原因：忽略了0的0次幂无意义

正确解法

正确：a⁰ = 1 (a ≠ 0)

提醒：0的0次幂没有意义

易错点：相似三角形 vs 全等三角形

错误：相似三角形就是全等三角形

错误原因：混淆了相似和全等的概念

正确区分

相似三角形：对应角相等，对应边成比例

全等三角形：对应角相等，对应边相等（相似比为1）

易错点：轴对称 vs 中心对称

错误：轴对称图形就是中心对称图形

错误原因：混淆了两种对称方式

正确区分

轴对称：关于某条直线对称，折叠后重合

中心对称：关于某个点对称，旋转180°后重合

易错点：一次函数k的正负对图像的影响

错误：k为负时，图像从左到右上升

错误原因：记错了k的正负与图像走势的关系

正确理解

k>0：图像从左到右上升，y随x增大而增大

k<0：图像从左到右下降，y随x增大而减小

易错点：二次函数开口方向与a的关系

错误：a为负时，抛物线开口向上

错误原因：记错了a的正负与开口方向的关系

正确理解

a>0：抛物线开口向上

a<0：抛物线开口向下

易错点：概率中“放回抽样” vs “不放回抽样”

错误：两种抽样方式概率计算方法相同

错误原因：忽略了抽样方式对概率的影响

正确区分

放回抽样：每次抽样的概率不变

不放回抽样：每次抽样的概率会变化

例：从10个球中抽2个，放回时概率为(1/10)×(1/10)，不放回时为(1/10)×(1/9)

5. 题型分类题库

5.1 代数

整式运算

基础

计算：(2x + 3)(x - 4)

解析：

步骤1：使用分配律展开

(2x + 3)(x - 4) = 2x·x - 2x·4 + 3·x - 3·4

步骤2：计算每一项

= 2x² - 8x + 3x - 12

步骤3：合并同类项

= 2x² - 5x - 12

答案：2x² - 5x - 12

分式化简

中档

化简：(x² - 4)/(x + 2)

解析：

步骤1：因式分解分子

x² - 4 = (x + 2)(x - 2)

步骤2：约分

(x + 2)(x - 2)/(x + 2) = x - 2 (x ≠ -2)

答案：x - 2 (x ≠ -2)

5.2 几何

三角形全等证明

中档

已知：在△ABC和△DEF中，AB=DE，BC=EF，∠B=∠E，求证：△ABC≌△DEF

解析：

证明：在△ABC和△DEF中

AB=DE（已知）

∠B=∠E（已知）

BC=EF（已知）

根据SAS（边角边）判定定理，△ABC≌△DEF

答案：证明如上

求角度

基础

在直角三角形中，一个锐角为30°，求另一个锐角的度数。

解析：

直角三角形的两个锐角和为90°

另一个锐角 = 90° - 30° = 60°

答案：60°

5.3 统计

平均数计算

基础

计算数据：2, 4, 6, 8, 10 的平均数。

解析：

平均数 = (2 + 4 + 6 + 8 + 10)/5 = 30/5 = 6

答案：6

概率计算

中档

从一副52张扑克牌中随机抽取一张，求抽到红桃的概率。

解析：

一副扑克牌有52张，红桃有13张

概率 = 13/52 = 1/4

答案：1/4
